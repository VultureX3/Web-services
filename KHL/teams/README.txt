Данный веб-сервис сделан на платформе фреймворка Django с использованием REST API.

Необходимые модули Python для работы сервиса: Django, Djangorestframework

Сервис позволяет создавать команды и добавлять в них игроков. Все данные содержатся в базе db.sqlite3.
Там же находится и лог сервиса, в котором содержатся 4 графы: ip пользователя, url, время действия и его успех (неуспехом считается запрос, не прошедший валидацию).

Запуск:
1. В командной строке в папке teams ввести python manage.py runserver
2. В браузере ввести http://127.0.0.1:8000
