Данный веб-сервис сделан с помощью фреймворка bottle.

Необходимый модуль Python для работы сервиса: bottle

Сервис представляет из себя небольшой файловый менеджер с корнем в папке программы.

Запуск:
1. В командной строке в папке File_manager ввести python File_manager.py
2. В браузере ввести http://localhost:8080
